# Quick Sort
피벗을 이용하여 정렬하는 알고리즘
## Detail
`리스트` 중 하나의 원소를 고른다. 이를 `피벗(pivot)`이라 부른다.<br/>
`피벗`을 기준으로 `피벗` 앞은 `피벗`보다 작은 수가 오고, 뒤는 `피벗`보다 큰 수가 온다.<br/>
`피벗`을 기준으로 나누어 `재귀`적으로 반복한다.<br/>

## Time Complexity
Best - O(nlog2(n))
Worst - O(n^2)
Avg - O(nlog2(n))

## Advantage / Disadvantage
### Advantage
- 속도가 제일 빠르다.
- 추가적인 메모리 공간을 필요로 하지 않는다.
### Disadvantage
- 정렬된 리스트에 대해서는 오히려 속도가 느리다.
