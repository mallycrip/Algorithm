# Merge Sort
분할 하여 문제를 해결하고 결과를 모아서 정렬하는 알고리즘
## Detail
리스트를 __두 개의 균등한 균등한 크기__ 로 분할 후, `분할 된 부분 리스트`를 정렬한 다음, `두 개의 정렬 된 부분 리스트`를 합하여 전체가 정렬된 리스트가 되게 하는 방법.

### Example
__8, 2, 4, 1, 3, 7, 5, 6__<br/>
<br/>
__8, 2, 4, 1__<br/>
<br/>
__8, 2__<br/>
`8` | `2` -> 2 <br/>
`8` | -> 2, 8 <br/>
Result = 2, 8<br/>
<br/>
__4, 1__<br/>
`4` | `1` -> 1<br/>
`4` | -> 1, 4<br/>
Result = 1, 4<br/>
<br/>
`2`, 8  |  `1`, 4 -> 1 <br/>
`2`, 8  |  `4` -> 1, 2 <br/>
`8`  |  `4` -> 1, 2, 4 <br/>
`8`  |  -> 1, 2, 4, 8 <br/>

Result = 1, 2, 4, 8



<br/>
__3, 7, 5, 6__<br/>
<br/>
__3, 7__<br/>
`3` | `7` -> 3 <Br/>
| `7` -> 3, 7<br/>
Result = 3, 7<br/>
<br/>
__5, 6__<br/>
`5` | `6` -> 5 <br/>
| `6` -> 5, 6<br/>
Result = 5, 6<br/>
<br/>
`3`, 7 | `5`, 6 -> 3 <br/>
`7` | `5`, 6 -> 3, 5 <br/>
`7` | `6` -> 3, 5, 6 <br/>
`7` | -> 3, 5, 6, 7 <br/>
Result = 3, 5, 6, 7<br/>
<br/>
`1`, 2, 4, 8 | `3`, 5, 6, 7 -> 1<br/>
`2`, 4, 8 | `3`, 5, 6, 7 -> 1, 2<br/>
`4`, 8 | `3`, 5, 6, 7 -> 1, 2, 3<br/>
`4`, 8 | `5`, 6, 7 -> 1, 2, 3, 4<br/>
`8` | `5`, 6, 7 -> 1, 2, 3, 4, 5<br/>
`8` | `6`, 7 -> 1, 2, 3, 4, 5, 6<br/>
`8` | `7` -> 1, 2, 3, 4, 5, 6, 7<br/>
`8` |  -> 1, 2, 3, 4, 5, 6, 7, 8<br/>
<br/>
Result = 1, 2, 3, 4, 5, 6, 7, 8
